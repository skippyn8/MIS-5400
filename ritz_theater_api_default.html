Ritz Theater API Documentation

Here you will find information on using the Ritz Theater Ticket Sales api.

The data for this project is stored in the RitzSales database.
It consists of the ticket, employee and concessions sales information for the Ritz Theater for several months in 2017 and 2018.

ADDING Data

POST /api/rs/sales/tickets - Add a new ticket sales record to the database
POST /api/rs/sales/concession - Add a new concession sales record to the database
POST /api/rs/sales/employee - Add a new employee record to the database
POST /api/rs/sales/feature - Add a new feature record to the database
POST /api/rs/sales/sales - Add a new sales record to the database


GETTING Data

GET /api/rs/sales/tickets - Get all (up to 1,000 rows) from the Ticket_Sales table in the database.
GET /api/rs/sales/tickets - Get all records from the Ticket_Sales table in the database.
GET /api/rs/sales/tickets/gross_id - Get a single record by ID from the Ticket_Sales table in the database.
GET /api/rs/sales/concession - Get all records from the Concession_Sales table in the database.
GET /api/rs/sales/concession/con_id - Get a single record by ID from the Concession_Sales table in the database.
GET /api/rs/sales/employee - Get all records from the Employee table in the database.
GET /api/rs/sales/employee/emp_id - Get a specific employee from the database by emp_id
GET /api/rs/sales/feature - Get all records from the Feature table in the database.
GET /api/rs/sales/feature/feature_id - Get a specific feature from the database by feature_id
GET /api/rs/sales/sales - Get all records from the Sales table in the database.
GET /api/rs/sales/bo_report - Get all records from the BoxOfficeReports view in the database.
GET /api/rs/sales/bo_report/show_title/show_date - Get all records from the BoxOfficeReports view in the database for a single date.
GET /api/rs/sales/bo_report/show_title/first_date-last_date - Get records for a single show from the BoxOfficeReports view in the database for a date range.
GET /api/rs/sales/bo_report/show_title - Get all records from the BoxOfficeReports view in the database for a single title.
GET /api/rs/sales/bo_report/rating - Get all records from the BoxOfficeReports view in the database for a specific rating.
GET /api/rs/sales/bo_report/distributor - Get all records from the BoxOfficeReports view in the database for a single distributor.
GET /api/rs/sales/total_sales - Get all records from the TotalSales view in the database.
GET /api/rs/sales/sales_by_date - Get all records from the SalesByDate view in the database.
GET /api/rs/sales/CandySalesByDate - Get all records from the CandySalesByDate view in the database.


DELETING Data

Deleting should not be done under any circumstance. It is a violation of the database rules!

DELETE /api/rs/sales/employee/emp_id - Delete single row from the Employee table by emp_id
DELETE /api/rs/sales/tickets/gross_id - Delete single row from the Ticket_Sales table by gross_id
DELETE /api/rs/sales/concession/con_id - Delete single row from the Concession_Sales table by con_id
DELETE /api/rs/sales/feature/feature_id - Delete single row from the Feature table by feature_id
